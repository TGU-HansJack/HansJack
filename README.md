# 寒士杰克 Typecho 主题

## 主题信息
- 主题名称：寒士杰克
- 作者：寒士杰克
- 网址：https://example.com
- 定位：个人创作空间型博客（强调写作体验、个人表达与长期阅读）

## 当前版本目标
当前版本优先实现首页第一屏与站点基础框架：
- 顶部三段式导航（左站点名、中导航、右登录/头像）
- 首页首屏双栏（左欢迎文案、右网站头像）
- 日间/夜间背景色与背景图可配置
- 内容区域桌面宽度比例可配置（60% 到 100%）
- 移动端适配（手机宽度下自动使用 100%）

## 安装方式
1. 确保主题目录为：`/usr/themes/<theme-dir>`
2. 在 Typecho 后台进入「控制台 -> 外观」
3. 启用「寒士杰克」主题
4. 进入主题设置，按需配置背景、首页文案、导航链接等选项

## 已实现页面
- 首页/归档列表：`index.php`（首页含首屏 Hero）
- 文章页：`post.php`
- 独立页面：`page.php`
- 404 页面：`404.php`

## 目录结构（核心）
- `index.php` 主题入口与首页内容
- `functions.php` 主题配置项与辅助函数
- `header.php` 顶部区域与动态样式变量注入
- `footer.php` 页脚
- `style.css` 全站样式
- `功能清单.md` 功能进展清单
